Upgrade the stake table proxy to use StakeTableV2. Internally, first detect existence of proxy, then deploy StakeTableV2 Assumes:

  • the proxy is already deployed.
  • the proxy is owned by a multisig.

Returns the url link to the upgrade proposal This function can only be called on a real network supported by the safeSDK
